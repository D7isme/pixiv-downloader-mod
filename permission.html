それぞれの権限について(Explanation of Each Permission)

pixiv.net
Pixivから画像をダウンロードするために必要です。
This permission is necessary to download images from Pixiv.
pximg.net
Pixivから画像をダウンロードするために必要です。
This permission is necessary to download images hosted on Pixiv.
orca-soft.net
同期機能や不具合発生時のアラート表示などに必要です。
This permission is required for synchronization features and to display alerts when issues occur.
ダウンロードの管理(Manage your downloads)
Pixivからダウンロードを行うために必要です。
This permission is needed to manage and perform downloads from Pixiv.
declarativeNetRequest / declarativeNetRequestWithHostAccess
Pixivからの画像のダウンロード要求を処理するために必要です。
This permission is necessary to process requests for downloading images from Pixiv.
